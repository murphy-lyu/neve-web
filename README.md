# carry-legal

Public legal documents for [Carry](https://github.com/murphy-lyu) — an iOS travel packing-list app made by Luma Studio.

## Live URLs

- **Hub**: https://murphy-lyu.github.io/carry-legal/
- **Privacy Policy**: https://murphy-lyu.github.io/carry-legal/privacy/ ([中文](https://murphy-lyu.github.io/carry-legal/privacy/zh.html))
- **Terms of Service**: https://murphy-lyu.github.io/carry-legal/terms/ ([中文](https://murphy-lyu.github.io/carry-legal/terms/zh.html))

## Structure

```
.
├── index.html              # Hub page
├── privacy/
│   ├── index.html          # English Privacy Policy
│   └── zh.html             # 简体中文隐私政策
└── terms/
    ├── index.html          # English Terms of Service
    └── zh.html             # 简体中文服务条款
```

## Updating

1. Edit the relevant HTML file
2. Update the "Last updated" date in both the page heading and any references
3. Commit and push — GitHub Pages rebuilds within a minute
